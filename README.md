SFSControl (Supports 1.6)
SFSControl is an SFS module that enhances the playability of the game.
Enabling players to control the game through network requests
The functionality of this module includes but is not limited to
Get mod version
Get all rocket information in the current world
Get simplified information about the specified rocket
Get information about the specified planet
Get all planet information
Get other information about the specified rocket
Get the specified rocket information
Get console logs
Get task information (v1.4.0 invalid)
Get terrain data for the specified planet
Get game screen (needs to be enabled in settings)
Get module CORS information
Worldwide drawing graphics

Set the specified rocket throttle
Set the specified rocket RCS
Activate specified rocket stage
Set the specified rocket SAS (rotation)
Display in-game prompt messages
Stop designated rocket rotation (not recommended)
Add stages to the specified rocket
Remove stages from the specified rocket
Launch rocket
Switch rocket
Rename specified rocket
Set navigation target
Clear navigation target
Increase time acceleration
Decrease time acceleration
Set time acceleration
Waiting until
Set cheat mode
Backtrack
Set the specified rocket's main engine on/off
Set the specified rocket orbit
Delete the specified rocket
Output log message
Complete the challenge
Track designated rocket
Switch map view
Unfocus
Transfer fuel of designated rocket
Stop designated rocket fuel transfer
Quick archive management
Control designated rocket wheels
Set designated rocket map icon color
Create rocket
Load blueprint
Switch to construction scene
Clear blueprint
